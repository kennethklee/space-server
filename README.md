space-server
============

[![Build Status](https://drone.io/github.com/kennethklee/space-server/status.png)](https://drone.io/github.com/kennethklee/space-server/latest)

Top down multiplayer space game server

DEFAULT PORT: 3000

orientation
-----------

```
       +y
       ^
       |
-x <---+--> x
       |
       \/
       -y
```
x increments right
y increments up
